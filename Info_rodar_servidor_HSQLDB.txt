Para rodar o servidor utilize o comando: 
java -cp hsqldb.jar org.hsqldb.server.Server --database.0 file:<FILE_NAME> --dbname.0 <DB_NAME>


Para abrir o gerenciador e se conectar ao banco de dados:
java -cp hsqldb.jar  org.hsqldb.util.DatabaseManager


Criando o banco de dados:
CREATE TABLE Produto (id INTEGER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY, nome VARCHAR(255), descricao VARCHAR(255))
